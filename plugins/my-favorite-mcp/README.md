# my-favorite-mcp

よく使う MCP サーバーをまとめたプラグインです。

新しいプロジェクトを始めるたびに「serena のインストールコマンドなんだっけ」と検索する作業から解放されます。その検索時間を積み上げると、おそらく映画一本分くらいにはなるでしょう。

## 含まれるサーバー

| Server   | 用途                             |
| -------- | -------------------------------- |
| serena   | シンボル単位でのコード解析・編集 |
| context7 | ライブラリドキュメントの取得     |

## 使い方

```bash
/plugin install my-favorite-mcp
```

その後、Claude Code を再起動してください。MCP サーバーが自動的に利用可能になります。

## サーバーを追加したい場合

`.mcp.json` を編集してください。形式は通常の MCP 設定と同じです。

追加したら、プラグインを再インストールするか、直接 `.mcp.json` をプロジェクトにコピーしてください。どちらでも動きます。どちらが良いかは哲学的な問題なので、ここでは触れません。
