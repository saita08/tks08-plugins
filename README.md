# tks08-plugins

Claude Code 用プラグイン集です。

同じ設定を何度もコピペする作業は、人生において最も創造性を発揮できる時間とは言えません。そこで、そういった作業を一度だけにするためのプラグインを作りました。結果として生まれた時間で何をするかは、あなた次第です。昼寝でも構いません。

## Plugins

各プラグインの詳細は `plugins/` ディレクトリをご覧ください。それぞれに README があります。読んでも読まなくても動きますが、読むと少しだけ幸せになれるかもしれません。

## Installation

```bash
# マーケットプレイスを追加
/plugin marketplace add https://github.com/saita08/tks08-plugins

# 何があるか確認
/plugin list

# 気になったものをインストール
/plugin install <plugin-name>
```

インストール後は再起動が必要です。これは仕様であり、バグではありません。多分。
